Polls (demo for aiohttp)
========================

Example of polls project using aiohttp_, aiomysql_ and aiohttp_jinja2_,
similar to django one.

Installation
============

Install the app::

    $ pip install -e .

Create database for your project::

    bash sql/install.sh

Run application::

    $ python -m app


Open browser::

    http://localhost:8080/

    :align: center


Requirements
============
* aiohttp_
* aiomysql_
* aiohttp_jinja2_
